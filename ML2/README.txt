This one is for an ice-skating competition. It implements the following:
o Type definition for a record (sinfo) which contains name, country and list of 5 judges’ scores for a skater.
o Type definition for a tuple (skater) which contains name, country and average score of a skater.
o A function which takes a list of records of type sinfo containing the information of skaters in a competition and converts them into a list of tuples of type skater with the average score.
o A function to return the top 2 skaters in the competition.
